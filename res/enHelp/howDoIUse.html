How do I use this program?

This program uses PlantUML language to create UML diagrams right from your java source code (only Java 7 and below is supported now). Output is PlantUML code, which can be interpreted by PlantUML software or plugins for different IDE's. Check out the PlantUML web-site to get more information. If you do not want to do anything further, our program and installed Graphviz software, are the only things that you need, to see diagrams of your Java project. If your java source code is too big and your computer is slow, we're recommending to you to turn off the option "Show diagram", and generate only PlantUML code. After that you could click the button "Generate via web". This will generate a diagram using PlantUML server and will open it on the PlantUML website. Below you will find instructions on how to create a chart using only this program.

  • First of all, if you want to see image of diagram in "Diagram" tab, you should install Graphviz software. It can be downloaded from Graphviz web-site. Installation process for different operating systems may vary.

  • Then you should choose directory of your Java project, or jar archive, by clicking the button "Select files".

  • Then, optionally, set preferred parameters of the future diagram. Such as: direction of diagram, shown relationships, type of files to be parsed, and preferred diagram type. Some of the parameters that you see may be unavailable for use. It simply means that we didn’t implement that functionality yet. Watch out for changes.

  • Press "Generate" button.

  • If all previous steps had been done correctly, you will see PlantUML code in "PlantUML code" tab, and in the ”Diagram" tab, you'll see UML diagram that represents your project.

If Graphviz was not installed correctly on your system, and you're trying to generate a class diagram, you will see an error message generated by PlantUML in the "Diagram" tab.
